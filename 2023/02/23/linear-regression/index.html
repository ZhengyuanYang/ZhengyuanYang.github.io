1. Linear Regression

1.1 Hypothesis Space and Basic Assumption

The Linear Regression is a statistical model used to discover the correlation between an output variable Y and a set of input variable X. Basically, a linear regression model should follow such hypothesis space:

Y=β1X+β0+ϵY = \beta _1X+\beta_0 +\epsilon Y=β1​X+β0​+ϵ

where β1\beta_1β1​ is a vector consists of parameters for each feature. β0\beta_0β0​ is a constant, ϵ\epsilonϵ represents the random error of the observation.

To modeling the data with a linear regression model, five assumption should be fulfilled:

1. Linearity

Linearity Assumption requires there to has a linear mapping relationship between Y and X. That is, the real mapping relationship should be consistent with the hypothesis space. Otherwise, there would be bais in the model.

2. ϵ\epsilonϵ Independency

The linear model assume the ϵ\epsilonϵ is a completely randomized error. That is, a sample’s ϵ\epsilonϵ shoud not be correlated with other samples. If this assumption cannot be satisfied, we say there exists autocorrelation in the data. Autocorrelation would make the model underestimate the randomized error and increase bias. It usually happened when there exists time-sequence influence among the samples.

3. X Independency

The features should be independent to each other, otherwise we said there exists multicollinearity. Multicollinearity would increase the variance of the model and cause overfitting

4.ϵ\epsilonϵ Homoskedasticity

The variance of ϵ\epsilonϵ should be same on different scale of Y. Otherwise, we said there is a Heteroskedasticity. Heteroskedasticity.increase the variance of the model and make the learned parameter unstable.

5. ϵ\epsilonϵ Normality

ϵ∼N(0,σ2)\epsilon \sim N(0,\sigma^2)ϵ∼N(0,σ2) . If the randomzied error are not normally distributed, the data points with high error would make the learned parameter unstable and increase the variances of the model. Notice that when ϵ∼N(0,σ2)\epsilon \sim N(0,\sigma^2)ϵ∼N(0,σ2), and Y=β1X+β0+ϵY = \beta _1X+\beta_0 +\epsilonY=β1​X+β0​+ϵ, we would have Y∼N(β1X+β0,σ2)Y \sim N(\beta _1X+\beta_0,\sigma^2)Y∼N(β1​X+β0​,σ2). This means (Y|X) should follows a normal distribution. Notice that this is not equivalent to Y∼N(μ,σ2)Y \sim N(\mu, \sigma^2)Y∼N(μ,σ2). We do not need to do normality test on Y before the learning.

The detection techniques and solutions for the violation of the above assumptions can be found in this article

Linear Regression is:

  • A supervised learning model
  • A parameter learning model
  • A non-probability learning model
  • A discriminative model

2. Regularization: Lasso Regression and Ridge Regression

A common problem in machine learning is the overfitting. For the details of overfitting, refer to this article

To solve overfitting, we can apply the structural Risk Minimization. We would add a penalty term regarding to the complexity of the model to the loss function. For details, refer to this article

The application of SRM on linear regression create Lasso and Ridge Regression. The Lasso and Ridge Regression have same training process like linear regression, but it add a regularization term of ω\omegaω in the loss function to penalize overfitting:

Lasso Regression

J=MSE+λ∣∣ω∣∣1J = MSE + \lambda ||\omega||_1 J=MSE+λ∣∣ω∣∣1​

Ridge Regression

J=MSE+λ∣∣ω∣∣22J = MSE + \lambda ||\omega||_2^2 J=MSE+λ∣∣ω∣∣22​

where λ\lambdaλ is a penalty parameter decides the extent of penalty. The regularization term is a L-P norm of the parameter vector. It is a generalized distance concept. Refer to the Minkowski Distance

The basic function of Lasso Regression and Ridge Regression are both minimizing the parameters(complexity), but they have the following differences:

  • The Lasso regression can reduce ωi\omega_iωi​ to 0, thus can be used for feature selection.
  • The Ridge mostly only reduce the parameter close to 0, thus is not very suitable for feature selection. We can still try it when the features are scaled though.

3. Logistic Regression

3.1 Basic Assumption

Logistics Regression is a model for classification. Unlike normal linear model, which assume the output is continuous and normally distributed, the logistic regression assume the output Y follows a Bernoulli Distribution.

The hypothesis space if logistic regression:

Let Z denote the linear transformation contributed by the a linear regression model:

Z=β1X+β0+ϵZ = \beta _1X+\beta_0 +\epsilon Z=β1​X+β0​+ϵ

Use an activation function sigmoid to convert such a transformation into a probability P(Y∣X)P(Y|X)P(Y∣X). For activation functions, refer to this article. The decision boundary is thus:

P(Y=1∣X=x)=sigmoid(Z)P(Y=1|X=x) = sigmoid(Z) P(Y=1∣X=x)=sigmoid(Z)

Decide the classification of Y by

ypred=argmax{P(Y=1∣X=x),P(Y=0∣X=x)}y_{pred} = argmax \{P(Y=1|X=x) , P(Y=0|X=x)\} ypred​=argmax{P(Y=1∣X=x),P(Y=0∣X=x)}

The Logistic Regression model is a discriminative probably model, as it directly modeling on P(X∣Y)P(X|Y)P(X∣Y)

Natural EFD

The Natural EFD is a subset of the EFD. If T(x) = x, we call this distribution a natural EFD. For example, Gaussian Distribution, Bernoulli Distribution, Exponential Distribution, Poisson Distribution, Gamma Distribution and Multinomial Distribution are all natural EFD.

4.2 Generalized Linear Model

A Generalized Linear Model is a model used to predict output Y with input X. Thus, a good idea is to estimate E[Y∣X]E[Y|X]E[Y∣X]. For example, the prediction of a linear regression is E[Y∣X]E[Y|X]E[Y∣X], the prediction of a logistic regression is E[Y∣X]=0∗P(Y=0∣X)+1∗P(Y=1∣X)=P(Y=1∣X)E[Y|X] = 0*P(Y=0|X) + 1*P(Y=1|X) = P(Y=1|X)E[Y∣X]=0∗P(Y=0∣X)+1∗P(Y=1∣X)=P(Y=1∣X), and the prediction muti-category regression(softmax regression) is E[Y⃗∣X,π⃗=p⃗]=[P(Y=0∣X),P(Y=1∣X),P(Y=k∣X)]E[\vec{Y}|X,\vec{\pi}=\vec{p}] = [P(Y=0|X),P(Y=1|X),P(Y=k|X)]E[Y∣X,π=p​]=[P(Y=0∣X),P(Y=1∣X),P(Y=k∣X)].

Now we nake the following assumption:

  1. The variable (Y|X) follows a natural EFD. Usually this can be guarantee by knowing Y follows a EFD.
  2. The canonical parameter of the EFD θμ\theta_\muθμ​, in a Bayesian context, follows a normal distribution, and thus can be learned from a linear hypothesis space

To achieve the expectation, a GLM consists of:

  • A linear predictor η=βX+b\eta = \beta X+bη=βX+b
  • A function that can transfer the linear prediction η\etaη to the expectation μ\muμ we want. μ=g−1(η)\mu = g^{-1}(\eta)μ=g−1(η)

We know that Y does not necessary follows a normal distribution and can be difficult to learned from with a linear hypothesis space. Thus we add a non-linear function to enhance the non-linearity of the hypothesis space. This function is called Activation Function, and the inverse of it η=g(μ)\eta = g( \mu)η=g(μ) is called a joing function.

We know from the assumption that θ\thetaθ is normally distributed and can be learned directly from the linear predictor. Thus, a good idea is to let g=ψ−1g = \psi^{-1}g=ψ−1, so we have:

η=g(μ)=ψ−1(μ)=θμ\eta = g(\mu) = \psi^{-1}(\mu) = \theta_\mu η=g(μ)=ψ−1(μ)=θμ​

In this case, the η\etaη we are learning is exactly θμ\theta_\muθμ​, which can be learned with a linear hypothesis space.

Hence, we we want to build a GLM to predict a variable Y∣X∼natural EFD(θ)Y|X \sim natural \ EFD(\theta)Y∣X∼natural EFD(θ)

  1. Figure out the distribution type of Y, and rewrite it into a canonical form of EFD
  2. Read θμ=ϕ(μ),b(θμ)\theta_\mu = \phi(\mu), b(\theta_\mu)θμ​=ϕ(μ),b(θμ​) out of the format and write down μ=ψ(θμ)\mu = \psi(\theta_\mu)μ=ψ(θμ​)
  3. Select the activation function as ψ−1(η)\psi^{-1}(\eta)ψ−1(η)
  4. Train the model

The GLM enable us to generalize linear regression to many other cases, as long as Y|X follows a EFD. Common application includes Logistic Regression(Bernoulli Distribution), Softmax Regression(multi-category distribution), Polynomial Regression and so on. When Y|X follows a gaussian distribution, the activation function is just μ=η\mu = \etaμ=η, and the model is reduced to a simply linear regression, which Y can be directly learned from a linear hypothesis space.
