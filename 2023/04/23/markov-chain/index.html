Basics of Markov Chain

1. About Markov Chain

Markov Chain is a mathematical model used to describe a system’s behavior over time, where the future state of the system depends only on its current state. It is a stochastic process that is widely used in various fields such as physics, economics, and computer science.

A Markov Chain is a sequence of random variables, where each variable represents the state of the system at a specific time. The probability of transitioning from one state to another is based only on the current state of the system, and not on the past states.

Let XnX_ nXn​ denote a be a stochastic process that takes on finite possible values {0,1,...m}\{0,1,...m \}{0,1,...m}. For all i,j,ni, j , ni,j,n, if we have:

P{Xn+1=j∣Xn=i,Xn−1=in−1,...X0=i0}=P{Xn+1=j∣Xn=i}P\{X_{n+1} = j | X_n = i, X_{n-1} = i_{n-1},...X_0 = i_0\} = P\{X_{n+1} = j | X_n = i\} P{Xn+1​=j∣Xn​=i,Xn−1​=in−1​,...X0​=i0​}=P{Xn+1​=j∣Xn​=i}

Then we said this random process has Markov Property. We can P{Xn+1=j∣Xn=i}P\{X_{n+1} = j | X_n = i\}P{Xn+1​=j∣Xn​=i} the one-step transition probability from state iii to state jjj , denoted as PijP_ {ij}Pij​.

2.2 Classification of States

Communicate

State jjj is said to be accessible from state iii if Pijn>0P_{ij}^n > 0Pijn​>0 for some nnn . In other word, start from state iii , it is possible to ever enter state jjj . If two state are accessible from each other, we say these two states communicate. If iii communicate with jjj , j communicate with kkk , then iii communicate with kkk . We call a group of states {i0,i1,...}\{i_ 0,i_ 1,...\}{i0​,i1​,...} a class if every pair of states in the class communicates. If any two states of a markov chain communicates, then there is only one class of states. In this case, we call the markov chain irreducible.

Recurrent and Transient

We said a state iii is recurrent if ∑n=1∞Piin=∞\sum^ \infty _{n= 1} P^n_{ii} = \infty∑n=1∞​Piin​=∞, or is transient if ∑n=1∞Piin<∞\sum^ \infty _{n= 1} P^n_{ii} < \infty∑n=1∞​Piin​<∞. This can be more clearly understood from a transition diagram:

We say state iii is recurrent if, starting from state iii , wherever you go to, you can always find a way back to iii.

On the contrary, we sat state iii is transient if, starting from state iii, if you go to some where, you can never find a way back to iii .

  • If state iii is recurrent, and state i communicates with state jjj , then state jjj is recurrent.
  • If state iii is transient, and state i communicates with state jjj , then state jjj is transient.
  • Recurrent and transient are class properties. If one element if the class is recurrent or transient, then all elements in that class are recurrent or transient as well
  • At least one state in a Markov Chain should be recurrent
  • Let μii\mu_ {ii }μii​ denote the expectation of steps it takes from iii back to iii. If μii>0\mu_{ii } > 0μii​>0, we call it positive recurrent, if μii=0\mu_ {ii} = 0μii​=0, we call it zero recurrent.
  • If state iii is transient, then the number of times the process will reenter iii follows a geometric distribution Geo(1−fii)Geo(1-f_{ii })Geo(1−fii​), where fiif_{ii }fii​ is the probability that the process will ever reenter jjj from jjj .

Periodical

For a state iii, let ddd denote the Greatest Common Divisor of a collection of n that Piin>0P^n_{ii} > 0Piin​>0 (d=0d = 0d=0 if the collection is empty ). If d>1d > 1d>1, then we said this state is periodical. For example, a state is only possible to reenter itself when the step n=2,4,6,8,...n = 2,4,6,8,...n=2,4,6,8,.... The greatest common divisor is 2. which makes this state periodical. If a state is:

  1. aperiodic
  2. positive recurrent

we call this state ergodic. If all states in an irreducible markov chain is ergodic, then we call this markov chain a **ergodic chain **

2.3 Stationary Distribution and Balance Equations

Consider a markov chain go through infinite steps. Under some condition, we would find that:

The condition of being able to reach stationary distribution is complex, but basically, a markov chain will eventually converge if it satisfies:

  1. irreducible
  2. all states are aperiodic

Specifically, for a markov chain reach stationary distribution, let π=(π1,π2,...πm)\pi = (\pi_1,\pi_ 2,...\pi_ m)π=(π1​,π2​,...πm​) , P denote the one-step transition probability matrix, we should have:

Thus, we can find the long-run proportion by solving a system of equations. We call this system of equations the stationary balance equations.

Specifically, if the stationary distribution of a markov chain does not change when we reverse the time order, we call this markov chain time reversible. In this case we have:

πiPij=πiPjiπ⋅P=π⋅PT\begin{aligned} \pi_i P_{ij} &= \pi_i P_{ji}\\ \pi \cdot P &= \pi \cdot P^ T \end{aligned} πi​Pij​π⋅P​=πi​Pji​=π⋅PT​

It can be proven that a time reversible markov chain has to be able to reach stationary distribution.

If a markov chain has transient states, it is not irreducible and does not necessarily have a stationary distribution. In such case, we can still calculate the mean time spent in transient states. Such a mean is different given different start point. Since it is impossible to enter transient state from recurrent state, the start point can only be transient states.

3. Continuous Time Markov Chain

3.1 Transition Rate

We said a random process is a Continuous Time Markov Chain if for all moments and states, we have:

P{Xs+t=j∣X1=i1,...Xs=is}=P{Xs+t=j∣Xs=i}P\{ X_{s+t} = j | X_1 = i_1,...X_s= i_s\} = P\{ X_{s+ t} = j|X_s = i \} P{Xs+t​=j∣X1​=i1​,...Xs​=is​}=P{Xs+t​=j∣Xs​=i}

If this probability is independent to sss , we say this markov chain has a stationary transition probability, denoted as Pij(t)P_{ij}(t )Pij​(t). We can such markov chain a Time-homogeneous Markov Chain. For example, the Poisson Process is a time-homogeneous CTMC.

In a continuous time Markov chain, the transition times ttt between states are exponentially distributed.
