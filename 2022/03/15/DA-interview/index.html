DA Interview Question

DA Interview Question

本文主要梳理国内数据分析师岗位面试中常见的八股文和场景化问题

1. 场景化问题

1.1 异动指标分析

某一指标（DAU，GMV）在某一时间明显下降，试分析原因和解决办法

  1. 检查异动是否由统计错误引起

    1. 检验数据格式是否正确
    2. 检验数据极值
    3. 通过http请求构造数据日志，对比日志，服务器数据和历史数据，如果是大数据则可采样后再注入日志

  2. 将影响因子（指标）拆解为外部和内部，首先分析外部：

    1. 观察异动指标过去一段时间的趋势，是否本身就处于下降趋势中？
    2. 观察异动指标同期历史数据，是否该时期的指标本身就低？
    3. 观察当下各类时事，是否对指标产生冲击？（对手促销，热点内容，竞品上市）

  3. 观察内部因素，进一步拆分成若干个维度，例如：

    1. 逻辑拆解
    2. 用户新老
    3. 渠道（APP，PC）
    4. 内容
    5. 地域

  4. 观察不同维度上指标的分布，确定影响系数：

    例如，如果流失用户中新老用户的占比为8:2，则有理由认为用户新老是一重要特征。相对，分布平均的维度可能相对不重要

  5. 在影响系数较大的维度上继续拆分，并结合两级维度做出假设

    例如新用户在APP和PC断的流失数量为7:3，可以假设新用户不喜欢最近的APP前端设计

  6. 继续拆分，直到能给出某一个具体可解决的假设

    例如最终假设为，用户不喜欢APP界面上短视频的双排列模式

  7. 根据此笼统的假设设计实验，定义metrics，给出具体假设，进行假设检验

1.2 费米估计

例如，不参考任何资料，估计北京市有多少红色的小轿车

  1. 将原问题拆解为多个问题，或者多个因子相乘，直到大多数因子看起来可以用常识进行判断上下界

    北京市红色小车数量 = 北京市人口 * 拥车比例 * 城市因子（城市一般拥车率更高） * 红色比例

  2. 给出每个因子的上下界，并取均值作为数字

    例如常见的车色有8-10种，红色在中国比较受欢迎，估计红车比例为[1/8,1/4], 取均值约为0.18

  3. 因子相乘，获得答案

1.3 指标提升方案【施工中】

例如用户培养，用户增长的场景，制定一个计划

1.4 用户画像

首先进行用户人群的划分，一般为垂直用户和行业用户的比较

  1. 根据什么特征划分垂直用户？

    • 随机划分

    • 特殊时间段：例如某一时间段的用户转化率特别高

    • 平台划分：例如某一平台的指标特别高

  2. 研究用户的什么特征？

    • 静态特征：例如用户年龄，性别等和用户自身相关的特征
    • 动态特征：用户7日内的购买量，复购率，好评率等等
    • 用户行为：是否点赞，收藏，评论等等

  3. 如何比较用户特征：

    • 利用可视化的方法做图，比较得出特征的影响系数和相关性
    • 利用假设检验，检查影响系数和相关性是否显著
    • 利用模型，对特征间的关系进行建模，并比较特征差异和重要性，例如线性模型，聚类模型，树模型等

  4. 分析以上结果，最终给出有关用户特征的结论，例如：高转化率的人群一般年龄在a-b岁，x渠道上的复购率比其他渠道高等等

  5. 将以上结论形成市场策略（结合项目的具体目标，例如广告投放，个性化运营，上线新功能等）

2. 业务相关指标和概念

以下指标往往有交叉，并不严格属于某一个行业

2.1 视频行业

播放量

播完率：播放时长\(\ge\)视频时长的比例

会话时长：例如人均访问/观看时长

PV：用户行为的技术，包括点赞，转发，评论，也可以统计比例

KOL：关键意见领袖，例如带货主播，明星等等

挑选KOL的方法

  1. 计算可选KOL以往带相似产品的ROI，挑选其中较高的
  2. 对这些KOL的用户做用户画像和标签化
  3. 挑选和这些KOL用户群体相似的KOL

视频热度评价：

  1. 播放维度：近期播放量，播完率

  2. 主播维度：主播关注数量，主播关注近期增量

  3. 互动维度：近期点赞，收藏，转发，评论

  4. 时间维度：新鲜度，时事热点关联度

通过机器学习或者基于经验的方式确定这些指标的权重，计算热度

2.2 电商行业

ROI：投入回报率 \[ ROI = \frac{收入-成本}{投入} \] 具体计算方法因场景而异，例如在广告投放中

收入：一般为销售收入，如果是为了吸引潜在客户，还要加上潜在用户增加数*转化率

成本：一般包括生产成本和广告投放成本

影响广告投放ROI的因素包括：

  1. 投放平台的选择
  2. 广告设计
  3. 推广预算

GMV：商品成交金额，包含已付款，未付款，已取消，退款，拒守等等情况，某种意义上代表了电商平台的市场份额

ARPU：人均付费

ARPPU： 付费用户人均付费

复购率：一定时间内购买两次及以上的用户比例

传统电商与直播电商的区别：

传统电商 直播电商
营销 主要是娱乐，次要是营销
没有KOL，互动弱 有KOL互动强
价格稳定，出了特殊节日促销 价格不稳定。不同直播间不同，秒杀等
转化率低 转化率高

直播电商的指标：

供给侧：补货量，推广份额，推广接单量

需求侧：购买转化率，GMV，ROI，回购率，退单率

2.3 用户增长APP

DAU： 日活用户

留存率：某一日的新增用户在之后继续活跃的比例，包括次日留存，7日留存等等

转化率：用户通过某一商业转化阶段的比例，例如（通过广告进入商店的人数）/（看到广告的人数）

4. 简历深挖

4.1 回答思路

回答有关项目的问题，一定把逻辑说完整，从为什么想做这个项目，一直说道这个项目能给公司/产品带来什么好处，另外，在每个叙述中要尽量体现以下逻辑：

  1. 发现某问题
  2. 通过类似指标拆解的方法，给出一个解决方案
  3. 解决问题
  4. 验证解决办法是否有效，有没有更好的办法

4.2 其他常见问题（包括HR面）

  1. 你的项目有没有其他平行组，你的结果与他们相比如何

  2. 这个项目有没有做的不好的地方，如何改进

    最好主动想一想项目的缺点，回答的时候自己提一下有做的不好的地方，面试官很可能直接就问了

  3. 你跟更喜欢偏技术和工作还是偏业务的工作

    最好判断一下面试官是做哪方面的，跟着他的工作说

  4. 说一下你的优点和缺点

    优点提前编好即可，缺点要注意：

    • 该缺点不能是你从事工作的致命伤：例如事DA工作，你的缺点不能是逻辑不清晰，对数字不敏感等
    • 该缺点必须是可以和改正的
    • 不要说一些其实是优点的缺点，这样会让面试官觉得你自我认知不清晰
